📘 OpenAPI (Swagger) - Library API
🔍 Giới thiệu

OpenAPI Specification (OAS) là một chuẩn mô tả REST API phổ biến nhất hiện nay. Nó cho phép định nghĩa toàn bộ API (endpoint, request, response, schema) dưới dạng file YAML hoặc JSON.

OpenAPI thường được dùng cùng với Swagger UI để:

Tự động sinh giao diện test API

Generate client/server code

Làm tài liệu API chuyên nghiệp

📂 File trong thư mục

library.yaml: File mô tả API theo chuẩn OpenAPI 3.0

⚙️ Cài đặt & chạy
Dùng Swagger Editor (online)

Truy cập: https://editor.swagger.io/

Copy nội dung library.yaml vào

Xem preview API bên phải
