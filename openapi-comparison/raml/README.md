🔍 Giới thiệu

RAML (RESTful API Modeling Language) là một chuẩn do MuleSoft phát triển, dùng để mô tả REST API.

RAML tập trung vào:

Thiết kế API trước khi code (Design-first)

Tái sử dụng (reuse)

Modular hóa API
